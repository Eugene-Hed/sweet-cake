# Application mobile Sweet-Cake (à développer)

Ce dossier accueillera l'application mobile client.

## Technologies suggérées
- React Native / Expo ou Flutter
- Connexion au backend via `/api/v1`

## Endpoints clés pour le mobile
- `/api/v1/authentification/*` — Auth
- `/api/v1/categories` — Catalogue
- `/api/v1/produits` — Produits
- `/api/v1/commandes/*` — Commandes client
- `/api/v1/ateliers` — Ateliers
- `/api/v1/ateliers/:id/reservations` — Réservation
