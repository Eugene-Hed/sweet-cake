# Back-office Sweet-Cake (à développer)

Ce dossier accueillera le back-office web de gestion.

## Technologies suggérées
- Next.js ou Vite + React
- Connexion au backend via `/api/v1`

## Endpoints clés pour le back-office
- `/api/v1/authentification/*` — Auth admin/gestionnaire
- `/api/v1/utilisateurs/*` — Gestion utilisateurs
- `/api/v1/categories/*` — Gestion catégories
- `/api/v1/produits/*` — Gestion produits
- `/api/v1/commandes/*` — Gestion commandes
- `/api/v1/ateliers/*` — Gestion ateliers
- `/api/v1/reservations/*` — Réservations
- `/api/v1/articles-stock/*` — Gestion stock
- `/api/v1/tableau-de-bord/resume` — Dashboard
- `/api/v1/journaux-audit` — Audit
